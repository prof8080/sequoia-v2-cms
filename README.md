# Sequoia AI Insights - Static Site

هذا هو الإصدار الثابت (Static) من موقع Sequoia AI Insights، المصمم للاستضافة المجانية والدائمة على **GitHub Pages**.

## 🚀 كيفية النشر

الموقع مهيأ للنشر التلقائي عبر **GitHub Actions**. بمجرد رفع الكود إلى فرع `master` في مستودع GitHub الخاص بك، سيتم بناء الموقع ونشره تلقائياً.

## 📝 كيفية إضافة مقالات جديدة

بما أن الموقع أصبح ثابتاً، تتم إضافة المقالات عن طريق تعديل ملف البيانات:

1. اذهب إلى المسار: `client/public/data/articles.json`
2. أضف مقالاً جديداً بنفس التنسيق:
   ```json
   {
     "id": 3,
     "title": "عنوان المقال الجديد",
     "description": "وصف مختصر للمقال",
     "category": "تصنيف المقال",
     "published_at": "2025-03-24T12:00:00.000Z"
   }
   ```
3. قم بحفظ التغييرات ورفعها (Push) إلى GitHub. سيقوم الموقع بتحديث نفسه تلقائياً.

## 🛠 التقنيات المستخدمة

- **React 19**
- **Vite** (للبناء السريع)
- **Tailwind CSS** (للتصميم)
- **GitHub Actions** (للنشر التلقائي)

## 🌐 الرابط
سيكون الموقع متاحاً على الرابط: `https://<username>.github.io/sequoia-v2-cms/` (تأكد من تفعيل GitHub Pages في إعدادات المستودع).
