# Sequoia V2 - تعليمات التشغيل

هذه نسخة مُبسطة واحترافية من الموقع، تعتمد على React و SQLite وتتضمن لوحة تحكم للمشرف.

## هيكل المشروع
- `server/`: يحتوي على الكود البرمجي للخلفية وقاعدة البيانات (SQLite).
- `client/`: يحتوي على كود الواجهة الأمامية (React).

## الرفع على الاستضافة (cPanel / VPS)

1. ارفع كافة الملفات إلى المجلد الرئيسي على الاستضافة (عدا مجلد `node_modules`).
2. تأكد من وجود ملف `database.sqlite` (سيتم إنشاؤه تلقائياً عند أول تشغيل).
3. قم بتثبيت المكتبات وتشغيل الموقع عبر الأوامر:
   ```bash
   npm install
   npm run build
   npm start
   ```
4. **هام**: أدخل إلى الرابط `yoursite.com/admin` لإنشاء حساب المشرف والبدء في إضافة المقالات.

## ملاحظات هامة
- قاعدة البيانات موجودة في ملف `database.sqlite` في المجلد الرئيسي.
- تم دمج كود AdSense في `index.html` وملف `ads.txt` جاهز في مجلد `public`.
- لوحة التحكم متاحة على الرابط `/admin`.
